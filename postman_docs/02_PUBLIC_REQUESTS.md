# PUBLIC REQUESTS

## Get Categories

- Method: `GET`
- URL: `http://127.0.0.1:8000/api/categories/`

## Get All Items

- Method: `GET`
- URL: `http://127.0.0.1:8000/api/items/`

## Search Items

- Method: `GET`
- URL: `http://127.0.0.1:8000/api/items/search/?q=airpods&category=3&item_type=found`

## Get Item Detail

- Method: `GET`
- URL: `http://127.0.0.1:8000/api/items/12/`
